# ci-thesis-template
LaTeX Template of Thesis at Creative Informatics

```
├── cimt.cls
├── figs
│   ├── dj.jpg
│   ├── fig.eps
│   └── nowprinting.eps
├── latexmkrc
├── LICENSE
├── main.bib
├── main.tex
├── Makefile
├── README.md
└── src
    ├── ack.tex
    ├── appendix.tex
    ├── body.tex
    ├── concl.tex
    ├── eabst.tex
    ├── intro.tex
    ├── jabst.tex
    └── publications.tex
```

# How to use

1. Edit Tex files

2. Compile to draft pdf

    ```
    make
    ```

3. Show todo

    ```
    make todo
    ```
